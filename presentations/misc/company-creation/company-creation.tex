\slide{Typiske Managed Security Services}

\begin{quote}
\large In computing, managed security services (MSS) are network security services that have been outsourced to a service provider.
\end{quote}
Kilde: \link{http://en.wikipedia.org/wiki/Managed_security_service}

\begin{list1}
\item Opgaver som tidligere blev håndteret in-house, eller ignoreret:
\item Event opsamling og analyse, 
Email scanning, Anti-virus og spam,
\item Firewall opsætning, drift og konfiguration
\item Netværksopsætning internt, STP, RSTP, stacks, LACP, LLDP, ...
\item Netværksopsætning eksternt, BGP, LC-SC, single-mode, mono-mode, multi-mode, link-net, PI, PA, RIPE
\item Angreb DoS, DDoS m.v.
\end{list1}
