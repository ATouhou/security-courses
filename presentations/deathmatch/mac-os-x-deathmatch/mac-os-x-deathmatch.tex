\slide{Tiger new features}

\begin{list1}
  

\item UNIX
\begin{list2}
\item Spotlight Command Line Tools
Access Spotlight from a UNIX command line using mdls or mdfind.
\item Fine Grain Locking (SMP scalability)
Invite more threads to the party in your processor.
\item 64-bit Virtual Memory
Run the new generation of 64-bit applications that address massive
amounts of memory. 
\item Access Control Lists (ACL)
Go beyond the limitations of traditional UNIX file permissions and
enjoy greater flexibility over assigning access permissions to
individual users. 
\item HFS+ CLI file commands 
Use command line file commands on HFS+ items with proper
results. Utilities such as cp, mv, tar, rsync now use the same
standard APIs as Spotlight and access control lists to handle resource
forks. 
\item Korn Shell
Run scripts written for Sun Solaris more easily via AT\&T's ksh.
\item Remappable Modifier Keys
Remap modifiers such as control and caps lock for compatibility with
Windows and UNIX keyboard conventions 
\end{list2}
\end{list1}

Kilde:
\link{http://www.apple.com/macosx/newfeatures/over200.html}

\slide{Mac OS X sikkerhed} 

\begin{list1}
\item Basal UNIX sikkerhed som forventet
\begin{list2}
\item Almindelige filrettigheder
\item Almindelige brugere - root er deaktiveret som standard    
\end{list2}
\item derudover er integreret nogle gode applikationer
\begin{list2}
\item Keychain Access - indeholder nøgler og private data
\item FileVault - krypter hele dit hjemmekatalog nemt!
\item AES krypterede disk images - krypter filer og send som .dmg
  filer    
\item WPA og WPA2
\end{list2}
\item Firewallkonfigurationen via GUI er ikke noget værd!
\item {\bf NB: lidt farlig boot! kan nemt bootes til single-user!}
\end{list1}
